# 📊 Customer Churn Analysis Project

## 🚀 Project Overview

This project focuses on analyzing customer churn behavior using data
analytics techniques. The goal is to identify key factors influencing
churn and provide actionable insights.

------------------------------------------------------------------------

## 🛠️ Tech Stack

-   Python (Pandas, NumPy)
-   SQL (MySQL)
-   Tableau (Data Visualization)

------------------------------------------------------------------------

## 📂 Project Workflow

### 1. Data Cleaning (Python)

-   Handled missing values
-   Removed duplicates
-   Standardized column formats
-   Feature preparation for analysis

File: Cleaning.ipynb

------------------------------------------------------------------------

### 2. Data Storage & Analysis (SQL)

-   Imported cleaned dataset into MySQL
-   Performed KPI calculations:
    -   Total Customers
    -   Churn Rate
    -   Revenue Analysis
-   Created SQL Views for Tableau integration

------------------------------------------------------------------------

### 3. Data Visualization (Tableau)

-   Built interactive dashboard
-   Key insights:
    -   Churn vs Non-Churn comparison
    -   Revenue distribution
    -   Customer behavior patterns

File: Churn Project.twbx

------------------------------------------------------------------------

## 📊 Key Insights

-   Identified high-risk customers based on usage and satisfaction
-   Observed revenue impact due to churn
-   Provided data-driven insights for retention strategies

------------------------------------------------------------------------

## 🎯 Conclusion

This project demonstrates end-to-end data analysis: Data Cleaning → Data
Processing → Data Visualization
